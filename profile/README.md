# TreeIDE

Bem-vindo a **TreeIDE**, uma organizacao dedicada a criar ferramentas simples, abertas e uteis para desenvolvedores que querem planejar, visualizar e gerar estruturas de projetos com rapidez.

Nosso principal projeto e o **TreeIDE**, um editor leve de estruturas de diretorios que combina escrita em texto, visualizacao em arvore e geracao automatica de pastas e arquivos.

## O que fazemos

- Criamos ferramentas para melhorar o fluxo de trabalho de desenvolvedores.
- Valorizamos interfaces simples, diretas e faceis de entender.
- Mantemos projetos open source com foco em produtividade.
- Exploramos formas mais rapidas de transformar ideias de arquitetura em estruturas reais de projeto.

## Projeto principal

### TreeIDE

**TreeIDE** e um editor desktop open source para desenhar estruturas de projetos usando texto simples.

Com ele, voce pode:

- Escrever uma estrutura de pastas e arquivos em formato textual.
- Visualizar a estrutura em arvore em tempo real.
- Salvar e carregar arquivos `.tree`.
- Gerar automaticamente diretorios e arquivos no seu computador.
- Alternar entre tema claro e escuro.
- Usar atalhos de teclado para acelerar a edicao.

Site: [tree.immare.xyz](https://tree.immare.xyz/)

## Exemplo

```text
src/
    index.js
    styles.css
assets/
    images/
    icons/
README.md
```

O TreeIDE transforma esse texto em uma arvore visual e pode criar essa estrutura automaticamente para voce.

## Valores

- **Simplicidade:** boas ferramentas devem remover atrito.
- **Clareza:** a estrutura de um projeto deve ser facil de ler antes mesmo de existir.
- **Abertura:** software aberto cresce melhor com colaboracao.
